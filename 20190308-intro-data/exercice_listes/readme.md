# Exercice listes

Utilisez les métodes `.map`, `.filter`, `.find` et  `.reduce` sur la liste `villes` pour que l'objet retourné soit: 

```json
{
  "nomsDesVilles": [
    "Lausanne",
    "Yverdon",
    "Montreux",
    "Renens",
    "Nyon",
    "Vevey"
  ],
  "villesDePlusDe30000Habitants": [
    {
      "nom": "Lausanne",
      "population": 138905
    },
    {
      "nom": "Yverdon",
      "population": 30143
    }
  ],
  "nombreDHabitantsDYverdon": 30143,
  "sommeDuNombreDHabitants": 257018
}
```
