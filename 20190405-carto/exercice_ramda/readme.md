# Exercice `fetch` + `ramda`

Vous avez deux URL qui retournent du JSON.

1. Utilisez `fetch` pour télécharger les données
2. Utilisez `ramda` pour créer une liste

```
[
  {
    nom_utilisateur: 'Machin',
    ville: 'Truc',
    nom_companie: 'Bidule',
    titres_posts: [
      'Titre 1',
      'Titre 2',
    ]
  },
]
```
